# Multivariate-Analysis-on-the-Glass-Identification-Dataset
In this project, I used multiple multivariate analysis methods to examine the Class Identification dataset.
Principal Component Analysis, Multidimensional Scaling, Discriminant Analysis, and BACON approach were applied to the dataset using R!
