# Multivariate-Analysis-on-the-Glass-Identification-Dataset

In this project, I used multiple multivariate analysis methods to examine the Class Identification dataset combined in the "Youssefdat.xls" file.
Principal Component Analysis, Multidimensional Scaling, Discriminant Analysis, Cluster Analysis, and BACON approach were applied to the dataset using R!

## Analysis Used

Each Project I, II, and III uses specific methods on the same dataset:
Project I: Outlier Detection and Discriminant Analysis
Project II: Cluster-Based Analysis
Project III: PCA and MDS
